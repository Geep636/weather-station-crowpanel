Weather Crow

ESP32-S3 E-Paper Weather Display 5.79-inch

Install Firmware

Your browser doesn't support Web Serial.
Please use Chrome, Edge, or Opera on desktop.

Requirements

  • Chrome, Edge, or Opera browser
  • USB cable connected to your ESP32-S3
  • Windows, macOS, or Linux
  • OpenWeatherMap API key (free)

How to Install

  1. Connect your ESP32-S3 device via USB
  2. Click the Install Weather Crow button above
  3. Select the serial port for your device(e.g. USB Serial)
  4. Wait for the installation to complete (about 2 minutes)
  5. Your device will restart and start the WiFi setup

How to Setup

  1. When the first boot, the device will create a WiFi hotspot called WeatherCrow-Config. while hotspot is active, red-led light will be on.
  2. Connect to it and go to 192.168.4.1 to configure WiFi and your OpenWeatherMap API key (free).
  3. You will need to your latitude and longitude to get the weather data. check it here.
  4. After save settings, the device will restart and connect to your WiFi.
⚠️ Change settings: If you want to update the settings, press and hold the Menu Button on the device and push the Reset Button. you can release the menu button when the device screen shows setup instructions.
